## Requirements
The code has been tested under
* Ubuntu 20.04 + NVIDIA GeForce RTX 1080Ti (CUDA 12.2)
* PyTorch 1.12.1

## Environment Setup
### Setup anaconda environment
```
$ conda create --name DCKO python=3.7.6 -y
$ conda activate DCKO
$ pip install -r /path/to/your/requirements.txt
```
